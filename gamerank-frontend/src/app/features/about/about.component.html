01 — Projeto

Sobre o GameRank

O GameRank foi desenvolvido como um projeto de portfólio com o objetivo de demonstrar conhecimentos práticos em desenvolvimento full stack moderno, arquitetura de microsserviços, integração entre APIs e conteinerização com Docker.

A proposta do sistema é funcionar como uma plataforma de avaliação de jogos eletrônicos, onde usuários podem visualizar jogos cadastrados, consultar informações e adicionar avaliações com notas e comentários.

Durante o desenvolvimento, foram aplicados conceitos como APIs REST, integração frontend/backend, persistência NoSQL, microsserviços, componentes reutilizáveis e arquitetura escalável.
